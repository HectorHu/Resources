## 通用
普通模式，退出vim q

普通模式，写需要sudo权限的文件，`:W`

普通模式，冒号 `:`(不需要按shift)

普通模式到行首 `shift + h`

普通模式到行尾 `shift + l`

## NERDTree
快捷键：

打开目录  F2

在目录树查找上层目录 u

光标移动到文件位置后打开文件并新建tab t

光标移动到文件位置后在当前tab打开文件 回车

前往下一个tab `]t`

回到上一个tab `[t`

向前移动n个tab ,n

回到第一个tab `,[`

去往最后一个tab `,]`

## 注释
增加代码注释 `,cc`

取消代码注释 `,cu`

## easymotion快速跳转到光标后任意位置
向下 `,,j`

向上 `,,k`

向后 `,,w`

查找'e',快速跳转定位到某个字符位置 `,,fe`

## Ctrlp搜索文件
启动 `,p`

相当于mru功能 `,f`

启动后上下移动 `ctrl + j/k`
